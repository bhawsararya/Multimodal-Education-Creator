# Multimodal Education Creator (Optimized)

## Tech Stack
- Gemini Pro (LLM)
- Stable Diffusion Turbo (Fast Image Generation)
- FAISS Vector DB
- Streamlit UI

## Setup
1. pip install -r requirements.txt
2. Rename .env.example to .env and add Gemini API key
3. streamlit run app.py

## Notes
- First image load downloads ~1GB (sd-turbo)
- CPU friendly & fast
